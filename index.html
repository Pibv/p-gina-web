Primer Encargo

Consiste en una composición abstracta inspirada en las obras de Vera Molnár, donde se utilizaban diversas formas y múltiples colores dinámicos. Este primer diseño fue realizado con las funciones más básicas de p5.js, con distintas formas, lineas, colores, rellenos etc.

Segundo Encargo

Consistió en realizar un sketch donde se incorporan las variables, las condicionales y los bucles, Creando asi una animación constante.

Tercer Encargo

Consistió en crear una composición visual animada que utilizara transformaciones geométricas para generar un patrón complejo o un sistema de movimiento. Experimentando asi la combinación de diferentes transformaciones y sistemas de coordenadas para lograr efectos visuales impactantes.

Cuarto Encargo

Se genero un patrón recursivo, en el que habia que implementar algún tipo de interacción con el mouse que a la vez provocaba una animación.

Quinto Encargo

Se genero un diseño a partir de la programación de objetos, tratandose de una interacción constante de elementos.

Sexto Encargo

Se trata de una composición a partir de un sistema de particulas, donde se experimenta la atracción, la repulsión y/o segumiento.

Septimo Encargo

Consiste en construir una imagen a partir de variables ya estipulas, donde llame a todas estas variables y les di una tonalidad diferente.
